# UI_NEW_Learning_Platform

A frontend learning platform UI built using **Next.js**, **React**, **TypeScript**, and **Tailwind CSS**.

This project contains the frontend screens and UI flow for a gamified learning platform.

---

## How to Run This Project Locally

Follow these basic steps to run the project on your laptop.

---

## 1. Clone the Repository

```bash
git clone https://github.com/jainamdavda1-pixel/UI_NEW_Learning_Platform.git
2. Go Inside the Project Folder
cd UI_NEW_Learning_Platform
3. Install Dependencies
npm install
4. Run the Frontend
npm run dev
5. Open in Browser

After running the command, open:

http://localhost:3000
6. Sign Up and View Student Dashboard

To view the Student Dashboard:

Open the project in your browser.
Click on Sign Up.
Choose Continue with Google.
Complete Google sign-up/login.
After successful sign-up, you will be redirected to the Student Dashboard.

From the student dashboard, you can explore modules, subtopics, simulations, quizzes, flashcards, progress, XP, badges, and other gamified learning features.
